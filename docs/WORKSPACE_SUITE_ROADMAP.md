# Workspace Suite Roadmap (toward Google Docs-like capability)

## Principle
Do not clone Google Docs wholesale. Build the minimum set of collaboration/document/data features that directly improve venture operations.

## Phase 1 (now): Ops Docs + Tables + Media
- Rich markdown docs with venture tagging
- Version snapshots and restore
- Data tables (filter/sort/group)
- Media library explorer + collections

## Phase 2: Collaboration basics
- Comments on docs and records
- Mentions/assignments
- Presence indicator (who is viewing/editing)
- Change history timeline per doc

## Phase 3: “Docs Suite” features
- Block-based editor (headings, callouts, embeds)
- Inline slash commands
- Reusable templates (SOP, campaign brief, postmortem)
- Linked records (doc <-> task <-> media <-> post)

## Phase 4: Spreadsheet-grade ops
- Formula fields
- Views (table/kanban/calendar)
- Saved filters per venture
- CSV import/export + sync connectors

## Phase 5: Automation-native workspace
- AI co-author with policy guardrails
- NL command bar for all bot actions
- Approval workflows embedded in docs/tables
- One-click “run workflow from doc context”

## MVP target (next)
1. Add doc version restore endpoint
2. Add comments + assignment metadata
3. Add linked records panel in Workspace Lite
4. Add table view for automation backlog and approvals
