# GateReady™ — Vercel Deployment Guide

## Quick Deploy (5 min)

### 1. Install dependencies locally
```bash
npm install
```

### 2. Test locally
```bash
npm start
# → http://localhost:3000
```

### 3. Deploy to Vercel

**Option A — Vercel CLI (recommended)**
```bash
npm install -g vercel
vercel login
vercel --prod
```
Follow the prompts. When asked:
- Set up and deploy?
